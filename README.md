# Robertwritescode Tap

## How do I install these formulae?

`brew install robertwritescode/tap/<formula>`

Or `brew tap robertwritescode/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "robertwritescode/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
